# 603 Sealcoating

Standalone static website for 603 Sealcoating.

## Files

- `index.html` contains the page content and owner-editable text.
- `style.css` contains all styling and responsive layout rules.
- `script.js` contains the mobile menu and current year behavior.
- `images/` is reserved for project photos and future site images.

Open `index.html` in a browser to preview the site.
